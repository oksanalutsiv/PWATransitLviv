# Icons

All icon components live in `src/assets/Icons/`. Each is a React component accepting `size` (default `24`) and `className` props.

```ts
import { IconBus } from '@/assets/Icons'

<IconBus size={24} className={styles.icon} />
```

Icons inherit colour from `currentColor` — set `color` on a parent or the element itself.

        {component}
      </div>
      <span style={{ fontSize: 11, color: '#6B7280', textAlign: 'center', fontFamily: 'monospace' }}>{name}</span>
    </div>
  ))}
</div>

## Sizing

Icons accept a `size` prop (number, px). The default is `24`.

export const sizes = [16, 20, 24, 32, 48]

<div style={{ display: 'flex', alignItems: 'flex-end', gap: 24, padding: '16px 0' }}>
  {sizes.map(s => (
    <div key={s} style={{ display: 'flex', flexDirection: 'column', alignItems: 'center', gap: 6 }}>
      <IconBus size={s} />
      <span style={{ fontSize: 11, color: '#6B7280', fontFamily: 'monospace' }}>{s}px</span>
    </div>
  ))}
</div>

## Colour

Icons use `currentColor`, so they pick up any CSS `color` value.

<div style={{ display: 'flex', gap: 24, padding: '16px 0' }}>
  {['#6366F1', '#10B981', '#F59E0B', '#EF4444', '#6B7280'].map(c => (
    <IconBus key={c} size={28} style={{ color: c }} />
  ))}
</div>
